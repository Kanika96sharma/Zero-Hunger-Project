# Zero Hunger Project

## 📌 Description
This is a web-based project developed using Flask to reduce food waste and help needy people by connecting donors and receivers.

## 💻 Technologies Used
- Python (Flask)
- HTML, CSS
- SQLite

## 🚀 Features
- User login and registration
- Food donation system
- Simple interface

## 📂 Project Structure
- app.py
- templates/
- static/
- database.db

## 🎯 Objective
To minimize food wastage and support people in need.

## 📌 Future Scope
- Add live tracking
- Improve UI
